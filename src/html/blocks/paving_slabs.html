Укладка тротуарной плитки

Для начала давайте рассмотрим распространенные примеры укладки:

"Елочка"

"Плетенка"

"Случайная смесь"

"Круговой шаблон"

"Шахматный порядок"

Каждый из узоров поможет создать особое определенное настроение для двора, подчеркнет особенности и нюансы ландшафтного дизайна, дополнит колорит.
